One Way Data Binding

  • One way data binding is used to bind the value of a property in the component to the UI.

1. Interpolation

  • Interpolation is the simplest way to bind data to the UI. It is used to display the value of a variable in the template.

FirstName = {{ firstName }}

RollNo = {{ rollNo }}

CurrentDate = {{ currentDate }}

2. Property Binding

  • Property binding is used to bind the value of a property in the component to the UI using the square brackets to the property name.

3. Event Binding

  • Event binding is used to bind the value of a property in the component to the UI.

Two Way Data Binding

  • Two way data binding is used to bind the value of a property in the component to the UI and the UI the component.

  • The value of the property is updated in the component and the UI is updated automatically.
